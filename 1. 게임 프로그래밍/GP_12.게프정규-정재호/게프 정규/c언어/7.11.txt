\n : 줄바꿈 문자 (제어 문자)
void (공허하다 비어있다.) 
function (함수, 기능)
main: os 에서 제일 먼저 부르는 함수 
error : 심각한 오류 (안고치면 실행 x)
warring : 경미한 오류
컴파일 시간 오류: 문법오류
실행시간 오류 : 실행 중에 0으로 나누는 연산 오류
논리 오류: 논리적으로 원하는 결과가 나오지 않고 잘못된 결과가 나오는 오류
error C2143 : ; 없음
error C1071 : 주석 오류
주석: 컴파일러가 분석하지 않음 코드 설명을 올리는 방법
type name = data; 
변수는 메인 메모리에서 만들어짐 ( 저장 하지 않으면 삭제)
메모리에는 주소가 있다. 
변수: 변경이 가능한 data, 또는 data를 저장하는 공간
상수: 변경이 불가능한 data
리터럴 : 이름없는 상수
area : 넓이, 면적
자료형 : 종류를 표시, 크기를 나타내기도 함
short, int, char, long : 정수 데이터
double, float: 실수형 데이터(3.1415)/ 부동소수점형
char: 문자형 데이터(‘a’)
char 1, int 4, float 4, double 8
sizeof 함수로 크기를 알아낼 수 있다. 
unsigned : 부호를 표현 x, 값이 두배가 된다.
